About This Project

The BioQR Health System is an innovative project built to securely manage health data and provide temporary access to files using biometric authentication and QR codes.

Users can authenticate via their fingerprint on an Android device, and generate a one-time QR code that grants limited-time access to files stored in a MySQL database. This ensures security and controlled data sharing.

Key Features

  • 🔒 Fingerprint authentication
  • 📂 Secure file storage in MySQL
  • ⏳ Time-limited one-time QR codes
  • 📱 Android app for easy access
  • 🌐 Node.js + Express backend
